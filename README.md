# Human Resources Analytics Dashboard

An interactive Power BI dashboard for analyzing and optimizing 
HR processes built as part of the CodeAlpha Power BI Internship.

## Features
- Recruitment metrics tracking
- Employee turnover rate analysis
- Employee satisfaction visualization
- Performance data analysis
- Hiring needs forecasting

## Tools Used
- Power BI Desktop
- Power Query for data cleaning
- DAX formulas for KPI calculations
- Slicers and filters for interactivity

## How to View
1. Download the .pbix file
2. Open it in Power BI Desktop
3. Interact with the dashboard filters and visuals
